My primary machine is a Dell XPS laptop that I picked up from Fry's. I run Ubuntu with Qtile, a tiling window manager written in Python.

Most of my programming is done with Komodo Edit because I like having an editor with the same interface regardless of what operating system I happen to be using at the time.

I'm a big fan of Google and use a lot of their products, including Chrome, Gmail, and Reader, and Docs.
